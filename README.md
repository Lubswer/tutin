# Sistema de Reservas de Biblioteca - EPN

## Descripción
Aplicación web para gestionar la reserva de espacios en las bibliotecas de la Escuela Politécnica Nacional.

## Tecnologías
- React + Vite
- Node.js + Express
- Firebase / Supabase
